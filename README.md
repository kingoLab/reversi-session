# Reversi Session (v7.3) - Full-Immersive

漆黒の宇宙を背景に、ネオンブルーとゴールドの意匠を纏った、完全没入型のリバーシ（オセロ）ゲーム・シミュレーター。高知能AI対戦機能と、思考を加速させるオーディオ・ガジェットをブラウザ上にワンペインで構築。

## 主な機能

* **高知能AI対戦（VS Gemini）**: 
  Gemini APIと連携し、Flashから最強COMBOまで、アルゴリズムと大局観を組み合わせたプロフェッショナルな対戦が可能。
* **神の試練（TRIAL OF GOD / GOD QUIZ）**: 
  10石以上の大ピンチ時にのみ発動する救済システム。全100問のサバイバル防災・救急知識クイズに挑み、正解すれば「神の恩寵」として盤面の石がランダムで自色に反転。v7.3より、フルスクリーンを維持したままモーダル内で完結する完全没入仕様へアップデート。
* **GOD HAND（神の手）**: 
  プレイヤーのターン時に、ウルトラハードエンジンの最善手を1手だけシミュレートする強力なアシスト機能。

## 仕様上の重要な注記：ミニCDプレイヤーについて

本アプリの左翼に配された「ミニCDプレイヤー」は、現代のPC環境（光学ドライブの非搭載、ブラウザによるローカルセキュリティの制約）に配慮した**「仮想オーディオ体験（メタファー）」**としてデザインされています。

物理的なCDドライブや実際の光学ディスクを直接読み込むものではありません。ユーザー自身のローカルにあるお好みの音楽フォルダ、または複数の音声ファイルを「仮想アルバム」としてローディング（装填）することで、80年代の無骨なオーディオガジェットを操作する情緒と没入感を100%再現する仕組みになっています。
